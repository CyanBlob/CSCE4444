FB IDs can be found at http://findmyfbid.com/
Access tokens can be generated at https://developers.facebook.com/tools/explorer/

Returns the number of friends user with facebook id '$fb_user_id' has
https://graph.facebook.com/{$fb_user_id}/friends?access_token={$fb_access_token}
